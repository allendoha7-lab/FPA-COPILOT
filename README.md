# FPA Copilot

An AI-powered Financial Planning & Analysis (FP&A) assistant designed to automate data processing, handle stress testing, and streamline financial reporting workflows.

## 📁 Project Structure

The repository contains the following core components:

* **`app.py`**: The main entry point for the application, handling the user interface and core AI/copilot logic.
* **`make_data.py`**: A utility script used to generate, process, or clean financial datasets for analysis.
* **`requirements.txt`**: List of all Python dependencies and libraries required to run the application.
* **`run_copilot.command`**: A macOS shell script to quickly launch the application with a single click.
* **`sample_data.xlsx`**: A template spreadsheet demonstrating the expected data structure for financial modeling, budgeting, and actuals comparison.
